Features

  • Embed your Flickr photos easily on your site.
  • Full Javascript widget. No flash required.
  • Easy markup, no need to write lots of Javascript.
  • Clear CSS styling
  • Keyboard navigation

Quickstart

Step 1: Include jQuery on your page

Either use your own hosted version or use the Google CDN (recommended).

					<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.4.3/jquery.min.js"></script>

Note: although jQuery 1.4.3 is loaded here, this plugin should work with all future versions of jQuery (and will be patched if it doesn't). You can easily load the latest version of jQuery by changing 1.4.3 to 1. To learn more about Google CDN versioning click here.

Step 2: Include the Flickry plugin

Include the plugin in the <head> of your page:

					<script src="/flickry/js/jquery.flickry.min.js"></script>

Note: make sure the paths point to the location where you've uploaded the files.

Step 3: Create a div where the widget should come

					<div class="gallery" data-set-id="72157625374372176"></div>

Step 5: Initialize the Flickry plugin

Place this snippet of code in your main javascript file or include it between <script> tags in your document

					$(document).ready(function() { 
						$('.gallery').flickry();	
					});

Step 6: it works!

You should now see a widget for the set!

Options

Tag attributes

In the quickstart you saw you could pass the set id in the div tag. You can pass these attributes:

Option name Description
data-set-id The id of the set to load. You can find this value in the URL when you browse to a set on Flickr.
data-user-id The id of owner of a photo stream to load. You can find this id by using this tool: idgettr.com
data-search A string to search for. If you also pass a user-id then it will search this user's photos.
data-limit The amount of photos to show.

These are the settings you can pass the function call:

Option name Description
api An object containing an url and a key. Normally you should not change this.
thumbnailSize default: 25
The size of the thumbnails.
keyboardControl default: true
Whether to enable keyboard control.
keyboardOnHover default: true
If this is set to true the keyboard controls only work when the mouse is hovering the element.
setId
userId
search
limit		 See above
fadeDuration default: 250
The duration of the fade effect in milliseconds.
